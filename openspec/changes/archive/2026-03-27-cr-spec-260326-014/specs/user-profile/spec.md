## ADDED Requirements

### Requirement: 學員頁面顯示學習紀錄預覽
學員頁面（`/user/[spiritId]`）SHALL 顯示「學習紀錄」預覽區塊，展示最近 3 筆結業紀錄（課程名稱、結業日期）；若超過 3 筆則顯示「查看更多」連結，點擊導向 `/learning`。

#### Scenario: 有結業紀錄的學員
- **WHEN** 存取有 3 筆以下結業記錄的學員頁面
- **THEN** 頁面顯示「學習紀錄」區塊，列出所有結業紀錄

#### Scenario: 結業紀錄超過 3 筆
- **WHEN** 存取有超過 3 筆結業記錄的學員頁面
- **THEN** 頁面僅顯示最近 3 筆，並顯示「查看更多」連結

#### Scenario: 點擊查看更多
- **WHEN** 使用者點擊「查看更多」連結
- **THEN** 導向 `/learning` 頁面

#### Scenario: 無結業紀錄的學員
- **WHEN** 存取無結業記錄的學員頁面
- **THEN** 不顯示「學習紀錄」區塊
