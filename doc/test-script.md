## 情境一：牧師開課 → 種子講師結業 → 取得開課資格

### 前置條件
- 牧師（黃國倫）已結業啟動靈人 + 啟動豐盛，具備開課資格
- 種子講師（吳容銘）尚未結業

### 步驟

**1. 牧師登入並開課**

1. 以牧師帳號 `gorden@test.com` / `Student@1234` 登入
2. 進入「課程管理」
3. 點擊「建立課程邀請」
4. 填寫：
   - 課程名稱：啟動靈人（第一梯）
   - 課程類型：啟動靈人
   - 開課日期：（自訂）
   - 人數上限：（自訂，建議 5）
5. 送出，確認課程邀請已建立，取得邀請碼

**2. 種子講師加入課程**

1. 以種子講師帳號 `romen@test.com` / `Student@1234` 登入
2. 進入「加入課程」，輸入邀請碼
3. 確認加入成功，課程狀態顯示「已報名」

**3. 牧師標記種子講師結業**

1. 切換回牧師帳號
2. 進入「課程管理」→ 選擇對應課程
3. 找到吳容銘，點擊「標記結業」
4. 確認結業狀態更新為「已結業」

**4. 驗證種子講師取得開課資格**

1. 切換回種子講師帳號
2. 確認「我的課程」頁面顯示啟動靈人已結業
3. 確認系統顯示「可開課」或開課按鈕已解鎖

**預期結果：**
- [ ] 課程邀請建立成功
- [ ] 種子講師能以邀請碼加入課程
- [ ] 牧師能標記種子講師結業
- [ ] 種子講師結業後取得開課資格

## 情境二：種子講師開課流程

### 前置條件
- 種子講師（吳容銘）已完成情境一，取得啟動靈人開課資格

### 步驟

**1. 種子講師建立課程邀請**

1. 以種子講師帳號 `romen@test.com` 登入
2. 進入「課程管理」
3. 點擊「建立課程邀請」
4. 填寫：
   - 課程名稱：吳容銘 的 啟動靈人
   - 課程類型：啟動靈人
   - 開課日期：（自訂）
   - 人數上限：3
5. 送出，取得邀請碼

**2. 複製並分享邀請碼**

1. 確認邀請碼顯示於課程詳情頁
2. 記錄邀請碼（後續情境三使用）

**3. 查看課程狀態**

1. 確認課程列表顯示新課程
2. 課程狀態為「進行中」或「已開課」
3. 已報名人數顯示 0

**預期結果：**
- [ ] 種子講師能成功建立課程邀請
- [ ] 邀請碼正確產生並可複製
- [ ] 課程列表即時更新

## 情境三：學員註冊、登入、加入課程

### 步驟

**1. 新學員註冊**

1. 開啟系統登入頁
2. 點擊「註冊」或「我要報名」
3. 填寫：
   - 姓名：（自訂）
   - Email：（使用未建立帳號的信箱，如 `newstudent@test.com`）
   - 密碼：（符合規則：含大小寫+數字）
4. 送出，確認帳號建立成功
5. 登入後確認需要「修改密碼」提示（若 isTempPassword）

**2. 現有學員登入**

1. 使用現有學員帳號 `hilo@test.com` / `Student@1234` 登入
2. 確認登入成功，進入個人頁面
3. （若提示修改密碼，先完成修改）

**3. 加入課程**

1. 進入「加入課程」功能
2. 輸入情境二取得的邀請碼
3. 確認成功加入，顯示課程名稱與開課日期
4. 重複以不同學員帳號加入（測試人數上限）：
   - `joyce@test.com`
   - `tony@test.com`
   - `hilo@test.com`（第 4 位，超過上限 3 位）
5. 確認第 4 位加入時系統顯示「已達人數上限」或排入等候

**4. 查看「我的課程」**

1. 進入學員個人頁面
2. 確認課程列表顯示剛加入的課程
3. 狀態顯示為「已報名」

**預期結果：**
- [ ] 新學員可成功註冊
- [ ] 現有學員可正常登入
- [ ] 邀請碼加入課程功能正常
- [ ] 人數上限控制有效
- [ ] 「我的課程」正確顯示已報名課程

## 情境四：管理者功能介紹

### 登入管理後台

1. 以管理員帳號 `101@iwillshare.org.tw` / `Admin@1234` 登入
2. 確認進入管理後台（側邊欄顯示管理選單）

### 4-1 使用者管理

1. 進入「使用者管理」
2. 確認使用者列表顯示所有帳號（含角色、靈人 ID）
3. 搜尋功能：輸入「黃國倫」，確認篩選結果正確
4. 查看黃國倫的詳細資料（結業紀錄應顯示：啟動靈人、啟動豐盛）
5. 修改使用者角色：
   - 選擇吳容銘，角色由 `user` 改為 `admin`（或其他角色）
   - 確認儲存成功，角色更新即時反映

### 4-2 課程管理（全覽）

1. 進入「課程管理」
2. 確認所有課程邀請列表顯示（含建立者、報名人數、狀態）
3. 開啟任一課程，確認可查看學員名單
4. 測試「標記結業」功能（為任一已報名學員標記結業）
5. 確認結業後學員的靈人紀錄更新

### 4-3 課程目錄管理

1. 進入「課程目錄」設定
2. 確認現有課程目錄：
   - 啟動靈人（開放）
   - 啟動豐盛（開放）
   - 啟動靈人 3（未開放）
   - 啟動靈人 4（未開放）
3. 測試啟用/停用課程目錄項目
4. 確認先修課程關係正確顯示

### 4-4 白名單管理（若有）

1. 進入「白名單管理」（Google 登入白名單）
2. 確認 Email 白名單列表
3. 新增一筆白名單 Email，確認儲存成功
4. 刪除剛新增的 Email，確認移除成功

### 4-5 教會 / 單位管理

1. 進入「教會/單位管理」
2. 確認現有清單：101、心欣、Kua、全福會
3. 新增一個單位名稱，確認儲存
4. 排序調整測試

### 4-6 系統設定

1. 確認版本號顯示於側邊欄底部
2. 確認系統名稱、Logo 顯示正確

**預期結果：**
- [ ] 管理員可查看所有使用者
- [ ] 角色修改功能正常
- [ ] 全覽課程管理正常運作
- [ ] 課程目錄管理可啟用/停用
- [ ] 白名單 CRUD 功能正常
- [ ] 教會/單位管理 CRUD 正常
